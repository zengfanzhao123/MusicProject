# Vue-MyMusic

## 简单实现音乐播放器

## 下载依赖

```
npm install
```

### 启动项目

```
npm run serve
```

### 打包项目

```
npm run build
```
## 功能

1.播放音乐
2.搜索
3.MV
4.播放器
5.电台
6.歌单
7.最近播放
8.我喜欢的音乐
9.登录
10.每日推荐
11.歌手页面
